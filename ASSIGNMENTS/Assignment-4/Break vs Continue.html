Break vs Continue Statement Example

Break Statement

The break statement exits the loop entirely when a condition is met. 


for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break;
    }
    console.log(i);
}

Continue Statement

The continue statement skips the rest of the loop for the current iteration and continues with the next iteration. 


for (let i = 0; i < 10; i++) {
    if (i === 5) {
        continue;
    }
    console.log(i);
}
